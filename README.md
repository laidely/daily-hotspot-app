# Daily Hotspot App — 每日热点洞察

> 基于AI驱动的每日热点新闻分析、圆桌研讨、效率工具与RAG方案的自动化工作流

## 📋 项目概述

本项目是一个完整的自动化工作流系统，从热点新闻检索到应用开发的全链路自动化执行，包含：

- **热点新闻分析**：全网检索当日科技/AI/商业领域热点
- **圆桌会议研讨**：模拟行业名人视角进行痛点深度分析
- **H5移动应用**：AI热点洞察移动端应用
- **SEO提示词模板**：基于CoT推理链的深度文章生成模板
- **RAG系统方案**：企业知识检索增强生成落地方案

## 📁 项目结构

```
daily-hotspot-app/
├── app/                          # H5移动端应用
│   ├── index.html                # 主页面（单页应用）
│   └── README.md                 # 应用说明
├── reports/                      # 分析报告
│   └── daily_analysis_20250530.md # 每日痛点分析报告
├── templates/                    # 提示词模板
│   └── seo_prompt_template.md    # SEO深度文章生成模板
├── docs/                         # 技术文档
│   └── rag_solution.md           # RAG系统落地方案
├── .gitignore
└── README.md
```

## 🚀 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **设计**：响应式移动端优先，深色科技主题
- **可视化**：Canvas API（雷达图）
- **AI/ML**：RAG架构（LangChain + Qdrant + DeepSeek）
