# AI热点每日速递

面向科技从业者的AI新闻聚合与深度分析工具，基于2026年7月AI行业真实热点数据构建。

## 应用概述

「AI热点每日速递」是一款纯前端移动端H5应用，为科技从业者提供AI行业最新动态的聚合阅读体验。应用内置18条基于2026年7月真实AI行业热点事件的模拟新闻数据，涵盖AI技术突破、办公自动化、效率工具、开发者趋势四大核心板块。

## 核心功能

### 1. 热点新闻卡片流
- 今日热榜排行榜（Top 5 热门新闻）
- 全部新闻信息流，支持按分类筛选
- 四大分类：AI技术 / 办公自动化 / 效率工具 / 开发者趋势
- 渐变色卡片设计，分类标识色彩区分

### 2. 深度分析
- 每条新闻支持展开查看详细内容
- 模拟生成多维度深度分析（技术分析、产业影响、投资视角等）
- 底部弹出式详情面板，支持下滑关闭

### 3. 个性化收藏
- 点击书签图标收藏感兴趣的新闻
- 收藏数据通过 localStorage 持久化存储
- 独立收藏页面查看所有已收藏新闻

### 4. 自定义标签
- 支持添加/删除个性化标签
- 标签数据通过 localStorage 持久化存储
- 预置4个常用标签：大模型、AI Agent、具身智能、多模态
- 最多支持20个自定义标签

### 5. 主题切换
- 深色/浅色主题一键切换
- CSS变量驱动的完整主题系统
- 主题偏好自动保存至 localStorage

### 6. 搜索功能
- 实时搜索新闻标题、摘要和标签
- 250ms 防抖优化
- 空结果友好提示

### 7. 相关推荐
- 新闻详情页底部展示同分类/同标签的关联新闻
- 支持在推荐新闻间连续浏览

## 技术特性

### 架构
- **纯单文件应用**：全部HTML/CSS/JS集中在 index.html 中，零依赖，无需构建工具
- **响应式设计**：以375px-428px手机宽度为主设计，同时适配更大屏幕
- **CSS变量主题系统**：通过 `data-theme` 属性和CSS自定义属性实现主题切换
- **无障碍支持**：语义化HTML标签、ARIA属性、键盘可访问

### 性能优化
- **IntersectionObserver图片懒加载**：卡片仅在进入视口时触发显示动画
- **虚拟滚动模拟**：滚动时仅处理可视区域附近的DOM元素
- **搜索防抖**：250ms延迟避免频繁渲染
- **CSS硬件加速**：动画使用 transform/opacity 触发GPU合成层
- **scroll事件节流**：使用 requestAnimationFrame 优化滚动监听

### 交互体验
- 流畅的CSS动画过渡（卡片渐入、面板滑出、主题切换）
- 触摸手势支持（下滑关闭详情面板）
- Toast提示替代原生alert
- 回到顶部浮动按钮
- 底部导航栏固定

### PWA支持
- 配置 manifest.json 支持添加至主屏幕
- 独立应用模式（standalone display）
- 竖屏锁定
- 主题色适配系统状态栏

## 使用方式

1. 直接在浏览器中打开 `index.html` 文件
2. 或通过本地HTTP服务器启动：
   ```bash
   # Python
   python3 -m http.server 8080 -d /workspace/app

   # Node.js
   npx serve /workspace/app
   ```
3. 在手机浏览器中访问对应地址，体验移动端效果
4. 可通过浏览器菜单"添加到主屏幕"获得类原生应用体验
