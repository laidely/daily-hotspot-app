# AI热点洞察

一款基于2025年5月30日热点新闻分析的移动端H5应用，采用科技感深色主题设计，提供AI驱动的新闻洞察与效率分析工具。

## 功能列表

### 首页 - 热点新闻卡片流
- 10条当日AI领域热点新闻，涵盖AI技术、办公自动化、效率工具、开发者趋势四大分类
- 分类标签筛选，支持横向滑动浏览
- 新闻卡片展示标题、摘要、阅读量、评论数、热门标识
- 滚动进入动画（IntersectionObserver）

### 洞察 - 圆桌会议观点
- 4位行业名人（李开复、Sam Altman、Andrej Karpathy、周鸿祎）的深度观点展示
- 每位专家配有头像、身份、核心观点引用、关键词标签
- 行业痛点雷达图可视化（Canvas绘制），展示5个维度的痛点评估

### 工具 - 效率计算器
- **AI Agent效率计算器**：输入每日工作时长、工作类型、AI工具熟练度，计算使用AI Agent后的效率提升数据
  - 支持软件开发、UI/UX设计、内容创作、数据分析、市场营销、项目管理6种工作类型
  - 三级AI熟练度（初学者/中级/高级）
  - 展示每日节省时间、效率提升百分比、等效工作时间、月均节省时间
  - 进度条动画可视化
- **每日热点摘要生成器**：模拟AI生成结构化摘要报告
  - 三种摘要风格：专业报告、轻松阅读、要点速览
  - 打字机效果逐字输出
  - 加载状态动画

### 我的 - 个人中心
- 个人信息卡片（头像、昵称、简介、统计数据）
- 功能菜单（收藏、历史、下载、设置、关于、反馈）
- 版本信息展示

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 语义化标签，移动端meta适配 |
| CSS3 | CSS变量、Flex/Grid布局、渐变背景、毛玻璃效果、CSS动画、过渡效果 |
| JavaScript (ES5) | 原生JS，SPA路由、Canvas绘图、IntersectionObserver、事件委托、requestAnimationFrame |
| Google Fonts | Noto Sans SC 中文字体 |
| Canvas API | 雷达图绘制与动画 |
| SVG | 内联图标（无外部图标库依赖） |

## 设计规范

- **移动端优先**：max-width 480px 居中布局，适配各种手机屏幕
- **配色方案**：深蓝黑底(#0a0e1a) + 青蓝主色(#00d4ff) + 紫色辅助(#8b5cf6) + 绿色强调(#10b981)
- **视觉效果**：渐变背景光晕、毛玻璃卡片、发光边框、流畅过渡动画
- **交互设计**：触摸滑动切换页面、底部Tab导航、分类标签筛选、Toast通知

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可运行
2. 推荐使用Chrome/Safari/Firefox等现代浏览器
3. 可通过浏览器开发者工具的设备模拟模式查看移动端效果
4. 所有功能均为前端本地运行，无需后端服务或网络请求（字体加载除外）

## 性能优化

- 图片全部使用CSS渐变/SVG内联替代，零图片请求
- 使用 `requestAnimationFrame` 驱动Canvas动画
- 事件委托统一管理点击事件，减少监听器数量
- IntersectionObserver实现懒加载动画，元素离开视口后取消观察
- CSS动画使用 `transform` 和 `opacity`，触发GPU加速
- 尊重 `prefers-reduced-motion` 系统偏好
